# 8x3 Priority Encoder

## Objective
Design and verify an 8x3 Priority Encoder using Verilog HDL.

## Tools Used
- Vivado Design Suite
- Verilog HDL
- Artix-7 FPGA

## Description
An 8x3 Priority Encoder converts eight input lines into a 3-bit binary code. If multiple inputs are active simultaneously, the highest-priority input is encoded.

Priority Order:
d7 > d6 > d5 > d4 > d3 > d2 > d1 > d0

## Result
The design was successfully simulated and verified in Vivado.
