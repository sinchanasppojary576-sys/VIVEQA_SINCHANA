# 3x8 Decoder

## Objective
Design and verify a 3x8 Decoder using Verilog HDL.

## Tools Used
- Vivado Design Suite
- Verilog HDL
- Artix-7 FPGA

## Description
A 3x8 decoder converts a 3-bit binary input into one of eight output lines. For each input combination, only one output line remains HIGH.

## Truth Table

| Input | Output |
|--------|---------|
| 000 | 00000001 |
| 001 | 00000010 |
| 010 | 00000100 |
| 011 | 00001000 |
| 100 | 00010000 |
| 101 | 00100000 |
| 110 | 01000000 |
| 111 | 10000000 |

## Result
The 3x8 Decoder was successfully designed, simulated, and verified in Vivado.
