# 4-bit Ripple Carry Adder

## Objective
Design and verify a 4-bit Ripple Carry Adder using four 1-bit Full Adders in Verilog HDL.

## Tools Used
- Vivado Design Suite
- Verilog HDL
- Artix-7 FPGA

## Description
A Ripple Carry Adder is constructed by cascading four 1-bit Full Adders. The carry output of each stage is connected to the carry input of the next stage.

## Result
The 4-bit Ripple Carry Adder was successfully designed, simulated, and verified in Vivado.
