# 4:1 Multiplexer using 2:1 Multiplexers

## Objective
Design and verify a 4:1 Multiplexer using three 2:1 Multiplexers in Verilog HDL.

## Tools Used
- Vivado Design Suite
- Verilog HDL
- Artix-7 FPGA


## Description
A 4:1 Multiplexer selects one of four input lines based on a 2-bit select signal and routes it to the output.

Selection:
- 00 → I0
- 01 → I1
- 10 → I2
- 11 → I3

## Result
The 4:1 Multiplexer was successfully designed using 2:1 Multiplexers and verified through simulation in Vivado.
