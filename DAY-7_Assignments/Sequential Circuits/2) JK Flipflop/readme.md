# JK Flip-Flop

## Aim
To design and verify a JK Flip-Flop using parameter declaration for HOLD, RESET, SET and TOGGLE operations.

## Operations

| J | K | Operation |
|---|---|-----------|
| 0 | 0 | HOLD |
| 0 | 1 | RESET |
| 1 | 0 | SET |
| 1 | 1 | TOGGLE |

## Tools Used
- Verilog HDL
- Xilinx Vivado
- Vivado XSim
