# VIVEQA_Sinchana_Digital_Design_assignment_Day_7

# Digital Logic Design Assignments

## About
This repository contains Verilog HDL assignments completed as part of Digital Logic Design and FPGA intership.

## Combinational Circuits

1. 1-bit Full Adder using Dataflow Abstraction
2. 2x4 Decoder using Dataflow Abstraction
3. 8x3 Priority Encoder using Structural Modeling
4. 4-bit Ripple Carry Adder using 1-bit Full Adder
5. 4:1 Multiplexer using 2:1 Multiplexers
6. 3:8 Decoder
7. 8:3 Priority Encoder

## Sequential Circuits

1. SR Latch using Gate-Level Modeling
2. JK Flip-Flop using Parameter Declaration
3. T Flip-Flop using D Flip-Flop
4. 4-bit Synchronous Loadable Binary Up Counter
5. 4-bit MOD-12 Loadable Binary Synchronous Up Counter
6. 4-bit Loadable Binary Synchronous Up-Down Counter

## Tools Used

- Verilog HDL
- Xilinx Vivado
- Vivado Simulator (XSim)
