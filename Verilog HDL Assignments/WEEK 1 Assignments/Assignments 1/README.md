# VIVEQA_VERILOG_HDL_FIRSTWEEK_ASSIGNMENT_1

3-Clock Cycle Delay Using Shift Register

Objective

To implement a delay circuit using a 3-bit shift register in Verilog HDL.

Description

The input signal is sampled at every positive edge of the clock and shifted through three register stages. The delayed output is taken from the final stage of the shift register.

Implementation

The design was implemented using:

* Verilog HDL
* 3-bit shift register
* Positive-edge triggered clock

Simulation

A testbench was created to verify the operation of the delay circuit. The waveform was analyzed to observe the propagation of data through the shift register stages.

Results

The design was successfully simulated and verified. The delayed output follows the input signal after passing through the register stages.

Files Included

* delay.v
* delay_tb.v
* waveform.png

Conclusion

The shift register delay circuit was successfully implemented and tested using Verilog HDL.
