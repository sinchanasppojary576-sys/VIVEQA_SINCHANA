# Verilog Tasks

Objective:
To study the implementation of Tasks in Verilog HDL.

Description:
A task named parity_cal is used to calculate the parity of a 32-bit input address. The task demonstrates:
- Task declaration
- Input and output arguments
- Delay statement inside a task
- Task invocation from an always block

Result:
The parity calculation was implemented using a Verilog task and simulated successfully.
